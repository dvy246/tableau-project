# 📊 Tableau End-to-End BI Project

## 🧩 Overview

This project demonstrates a complete **end-to-end Tableau Business Intelligence workflow** focused on analyzing **sales and customer performance for the year 2013**.

The dataset originates from an **internal data warehouse**, which was **cleaned, modeled, and visualized** using Tableau Desktop.  
It replicates a **real-world BI solution** that combines **data engineering**, **visual storytelling**, and **dashboard design principles** to deliver actionable business insights.

---

## 🎯 Objectives

- Connect and prepare data from a data warehouse  
- Create interactive dashboards for monitoring customer behavior and sales performance  
- Compare **year-over-year performance (2013 vs 2012)**  
- Identify **top customers**, **profit trends**, and **product category insights**  
- Apply design principles that enhance **clarity, interactivity, and decision-making**
